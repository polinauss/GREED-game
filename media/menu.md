# Menu

Аfter starting the game, the following options are available to the user:

# 1. Set player name
This option allows the player to enter a game name to save the result

# 2. Start new game
This option allows the player to start a new game

# 3. Load saved game
This option allows you to restore the game situation to a certain saved state (if present)

# 4. Game rules
This option allows the user to view game rules (in more detail [here](rules.md))

# 5. Leaderoard
Using this option, the player can match their attempts or compare the result with other players

# 6. Exit 
Аllows you to return to terminal input state
