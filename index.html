üìã About the Game

Greed is a console-based strategic game where players navigate a colorful grid, jumping on numbered cells to accumulate points while avoiding bombs and using teleporters for tactical movement.

üöÄ Quick Start

Requirements:

  • C++17 Compiler (g++ 7+ or clang++)
  • CMake 3.12+
  • Linux/Unix terminal with ANSI color support

Installation:

# Clone repository
git clone https://github.com/polinauss/GREED-game.git
cd greed-game

# Build with CMake
mkdir build && cd build
cmake .. && make

# Run the game
./greed_game
